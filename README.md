# vagrant-rig
vagrant box with Gitlab, Docker and Sonarqube

# Prerequisites
* Vagrant
* Virtualbox

# Building
make clean build

# Using
## SonarQube
http://localhost:9062

## Gitlab
http://localhost:8080

* NOTE: You still need to register the runner, which is a manual step.  See here: https://docs.gitlab.com/runner/install/linux-repository.html
